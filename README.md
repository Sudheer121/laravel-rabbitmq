# Laravel RabbitMQ

An easy-to-use Laravel package for working with RabbitMQ.

## Features

- Producers
- Consumers
- Publish / Subscribe
- Exchanges
  - Default
  - Direct
  - Topic
  - Fanout

## Requirements

- PHP 7.4+
- Laravel 5.5+

## Installation

```sh
composer require kunalvarma05/laravel-rabbitmq
```
